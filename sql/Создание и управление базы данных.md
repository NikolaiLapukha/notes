# Создание базы данных

При создании базы данных важно учитывать параметры, такие как кодировка символов (например, UTF-8) и порядок сортировки, чтобы обеспечить корректное хранение и обработку данных.

```sql
CREATE DATABASE имя_базы_данных
[CHARACTER SET кодировка]
[COLLATE сопоставление_символов];
```

- имя_базы_данных — это уникальное имя новой базы данных.
- CHARACTER SET — это параметр, определяющий кодировку символов, которая будет использоваться для хранения данных в этой базе данных. Например, "utf8mb4" — кодировка, поддерживающая широкий набор символов, включая эмодзи.
- COLLATE — это параметр, задающий порядок сортировки символов. Порядок сортировки влияет на то, как строки будут сравниваться в запросах (например, чувствительность к регистру при сортировке).
Пример:

```sql
CREATE DATABASE library
CHARACTER SET utf8mb4
COLLATE utf8mb4_general_ci;
```
# Выбор базы данных

После создания базы данных или если вы хотите работать с уже существующей базой, необходимо выбрать ее для дальнейших операций. Для этого используется команда `USE`.

```sql
USE имя_базы_данных;
```
# Удаление базы данных

Для удаления базы данных используется команда `DROP DATABASE`. Эта операция удаляет не только саму базу данных, но и все таблицы, индексы и данные, которые в ней содержатся. Это действие необратимо, поэтому перед выполнением данной команды стоит убедиться, что база данных больше не нужна или что у вас есть резервная копия.

```sql
DROP DATABASE имя_базы_данных;
```
