## Упаковка

```python
x, y = (1, 2)
# x = 1
# y = 2
a, *b = (2, 4, 3, 5)
# b = [4, 3, 5]
*x, y, z = 'hello python'
# x = ['h', 'e', 'l', 'l', 'o', ' ', 'p', 'y', 't', 'h']
# y = 'o'
# z = 'n'
```

## Распаковка

```python
a = [1, 2 ,4]
t = (*a,) # (1, 2, 4)
d = -5, 5 # (-5, 5)
[*range(*d)] # [-5, -4, -3, -2, -1, 0, 1, 2, 3, 4]
```

```python
arr = 'безнадежно убого неуд. удовл. хорошо отлично'.split()
a ={i:arr[i] for i in range(len(arr))}
print(a) # {0: 'безнадежно', 1: 'убого', 2: 'неуд.', 3: 'удовл.', 4: 'хорошо', 5: 'отлично'}
print(*a.items()) # (0, 'безнадежно') (1, 'убого') (2, 'неуд.') (3, 'удовл.') (4, 'хорошо') (5, 'отлично')
print(*a.values()) # безнадежно убого неуд. удовл. хорошо отлично
```
