# `find` - команда для поиска файлов

```bash
➜  ~ find Romeo*  
Romeo_and_Juliet_poem.txt
```

# `find -maxdepth <число>` - поиск с ограниченной глубиной

```bash
➜  ~ find -maxdepth 1  
.  
./.lmstudio-home-pointer  
./.cache  
./.local  
./.tmux  
./.gnupg  
./Romeo_and_Juliet_poem.txt  
./.ssh  
./.vim
...
```
