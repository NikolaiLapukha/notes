## Объявление переменных
Переменная - именованная ссылка на ячейку в память, которая хранит значение.

```python
a = 7
var = 'string'
b, c = 3, 4

```

## Моржовый оператор 

Позволяет присвоить значение переменной и вернуть это значение.

```python
print('YES' if len(a := input()) == len(set(a)) else 'NO')
```

```python
print(x := 5)
```
