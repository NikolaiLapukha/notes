Для создания таблиц в SQL используется команда `CREATE TABLE`, которая задает структуру таблицы, включая столбцы и их атрибуты.

```sql
CREATE TABLE название_таблицы
(
    название_столбца1 тип_данных атрибуты_столбца1,
    название_столбца2 тип_данных атрибуты_столбца2,
    ...
    название_столбцаN тип_данных атрибуты_столбцаN,
    атрибуты_уровня_таблицы
);
```

# Переименование таблиц

Если необходимо переименовать уже существующую таблицу, применяется команда `RENAME TABLE`.

```sql
RENAME TABLE старое_название TO новое_название;
```

# **Удаление таблиц**

Чтобы удалить таблицу вместе с её структурой и данными, применяется команда `DROP TABLE`.

```sql
DROP TABLE название_таблицы;
```

# Команда `ALTER TABLE`

В SQL для изменения структуры уже существующей таблицы используется команда `ALTER TABLE`. Она позволяет добавлять, изменять и удалять столбцы, а также управлять ключами и ограничениями.

```sql

ALTER TABLE имя_таблицы
{ 
    ADD имя_столбца тип_данных [атрибуты] |
    DROP COLUMN имя_столбца |
    MODIFY COLUMN имя_столбца тип_данных [атрибуты] |
    ALTER COLUMN имя_столбца SET DEFAULT значение |
    ADD [CONSTRAINT] определение_ограничения |
    DROP [CONSTRAINT] имя_ограничения 
};
```

**Пример:**

```sql
--Добавление нового столбца
ALTER TABLE Employees
ADD Email VARCHAR(100) NOT NULL;

--Удаление столбца
ALTER TABLE Employees
DROP COLUMN Email;

--Изменение значения по умолчанию
ALTER TABLE Employees
ALTER COLUMN Salary SET DEFAULT 50000;

--Изменение типа данных столбца
ALTER TABLE Employees
MODIFY COLUMN Name CHAR(60) NULL;

--Добавление внешнего ключа
ALTER TABLE Employees
ADD FOREIGN KEY (DepartmentID) REFERENCES Departments(DepartmentID);
```
